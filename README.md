# ClientCarePro

ClientCarePro is a customer service management web application developed using Spring Boot, Thymeleaf, MySQL, and Bootstrap.

## Features

- Customer Registration and Login
- Admin Login Dashboard
- Complaint Management
- Customer Feedback System
- Product Management
- News and Events Module
- Enquiry Management
- Password Recovery Module

## Technologies Used

- Java
- Spring Boot
- Thymeleaf
- MySQL
- Bootstrap
- Hibernate / JPA

## Modules

### Customer Panel
- Register/Login
- Submit Complaints
- Give Feedback
- View Products
- View News & Events

### Admin Panel
- Manage Customers
- Manage Products
- View Enquiries
- Manage Complaints
- View Feedback
- Add News & Events

## Project Setup

Clone repository:

```bash
git clone https://github.com/pratibhaksingh98/ClientCarePro.git
```

Run project using Spring Boot and configure MySQL in `application.properties`.
